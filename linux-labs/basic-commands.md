# Basic Linux Commands

## Commands Practiced

### ls
List files and directories.
```bash
ls
ls -l
``` 

### cd
Change directory.
```bash
cd /home
cd ..
```

### pwd
Show current directory
```bash
pwd
```

### mkdir
Create a directory
```bash
mkdir test-folder
```

### rm
Remove files or folder
```bash
rm file.text
rm -r foldername
```
